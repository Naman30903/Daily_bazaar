This is the **Admin Dashboard Webpanel** for the Daily Bazaar project.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Language**: TypeScript
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [shadcn/ui](https://ui.shadcn.com/)
- **State Management & API**: [React Query](https://tanstack.com/query/latest) & [Axios](https://axios-http.com/)
- **Charts**: [Recharts](https://recharts.org/)
- **Runtime**: [Bun](https://bun.sh/) (Recommended) or Node.js

## Getting Started

### Prerequisites

You can use **Bun** (recommended) or **npm/yarn/pnpm**.

### Installation

Navigate to the project directory:

```bash
cd admin_panel
```

Install dependencies:

```bash
# Using Bun
bun install

# Using npm
npm install
```

### Running the Development Server

```bash
# Using Bun
bun dev

# Using npm
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/`: Next.js App Router pages and layouts.
- `components/ui/`: Reusable UI components (shadcn/ui).
- `lib/`: Utility functions.
- `public/`: Static assets.
